Conectando Sistemas: The API Layer

protocol_rest.json

REST (Representational State Transfer)

O padrão por décadas. Utiliza métodos HTTP (GET, POST, PUT, DELETE) e códigos de status padrão. Stateless e cacheável por definição.

GET /api/v1/users/123
Host: mastery.io
Accept: application/json
query_graphql.gql

GraphQL (Graph Query Language)

Busca de dados declarativa. Os clientes solicitam exatamente o que precisam e nada mais. Elimina o over-fetching e o under-fetching.

query {
  user(id: "123") {
    name,
    email
  }
}

Quando Escolher?

STARTUPS

Escolha Monoliths. Velocidade de mercado é vital. Não desperdice ciclos iniciais gerenciando infraestruturas complexas até ter um produto provado.

ENTERPRISE

Escolha Microservices. Quando as equipes excedem 50+ engenheiros, o desacoplamento permite entregas autônomas e evita o travamento por coordenação.

DATA RICH

Escolha GraphQL. Se o seu frontend precisa agregar dados de múltiplas fontes em uma única viagem, a experiência do desenvolvedor é incomparável.
